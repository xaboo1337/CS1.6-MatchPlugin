### Match Plugin 5v5 CS 1.6

---

> [!IMPORTANT]
> Requirements: 
> - AmxModX 1.9.0-5271+
> - ReHLDS  v3.13.0.788+
> - ReGameDLL v5.26.0.668+
> - ReAPI v5.24.0.300+

## Install
1. Download latest release from [here]([https://github.com/ShadowsAdi/MixSystem/archive/main.ip](https://github.com/xaboo1337/CS-1.6-Match-Plugin.git))
2. Make sure the server runs on minimum requirements (run commands in server's console: 'version', 'game version', 'meta list').
3. Extract the *Match Plugin* archive and drag and drop files in your 'cstrike/addons/amxmodx' folder on your server (.amxx).

## This system includes the following functionalities:

- 100% customizable settings from an external file (from chat/console command names to player weapons during the warm-up round).
- Commands for:
  - Starting/stopping the frag.
  - Starting/stopping the mix.
  - Block/Unblock say chat
  - Block/Unblock M jointeam
  - Moving all players to spectator mode when player being frag.
